## INFO

- We highly recommend to stick to pre-installed libretro cores, which are used to emulate almost every possible system (via RetroArch) and have been optimized by our team for best possible performance. If there is a system missing for your favorite system, or there is compatibility problem with your specific ROM, then you may consider installing new (or updating existing) core via OnlineUpdater menu in RetroArch to use latest vanilla version
- For some edge cases, like performance drawback of using RA backend, you might try to install standalone emulator created by our community

## BIOS

- For all libretro cores please refer to [RetroArch documentation](https://docs.libretro.com/library/bios/)

- For standalone emulators see their instruction via installed Manual or in the respective repo's README
