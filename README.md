# MiyooCFW documentation pages

### Requirements:
```
pip install mkdocs
pip install mkdocs-material
```

### Preview:

- locally:
```
python3 -m mkdocs serve
```
- online:  
https://miyoocfw.gznetwork.com


### Build with
```
python3 -m mkdocs build
```
### Publishing

- Read _the_ Docs:  
Building & hosting is done via https://readthedocs.com
